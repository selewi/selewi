Details

Pixo runner was my first try at pairing up with someone to shape a game, and also, my first try at developing a mobile game. I meet a producer and we started working together to create this beautiful game about 'Pixo' the pixel boy, a tireless runner that needs help from the player to avoid obstacles.

Featuring 3 different levels, each one with unique mechanics, enemies and obstacles; Scoring system; A lot of runners to buy and customize with ingame coins, each one with it's own animations. This game teached me a lot of Unity tools and game development basics like object pooling, I also had to make many assets so I've learned a lot of sprite sheets, slicing and pixel art in general.
