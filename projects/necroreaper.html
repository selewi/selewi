Details

With only a few months in developing games, I decided to join my first game jam to make Necroreaper, I teamed up with an artist and a producer to create an ambitious 2D platformer where you must kill peasants to gather their souls. This game achieved the second place in the Game Dev Network: Gooseberry Jam.

The game was done in under 9 days from scratch, featuring fast paced action on a beautiful side scrolling map with many environments; solid AI for NPC's; an altar to get upgrades for the player which makes the gameplay gorgeous once you get them; a combo system which makes the player get a lot of score when doing succesive kills.
