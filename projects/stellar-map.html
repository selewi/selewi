Details

As I never made a 3D game before, I decided to create a simple prototype to get into it. Since I love space stuff I created a non scaled solar system map that features a smooth camera movement with zooming, real time orbits around the sun, a fictional travel system that uses bézier curves to represent the trajectory between markers and beautiful aesthetics implementing Unity particle system.
