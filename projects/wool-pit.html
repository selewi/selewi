Name: Wool pit

Team size: 3 persons

Available for:

Details

This game pushed my development skills to it's boundaries, I teamed up with two unknown persons to shape from scratch an arcade 2D top-down game about sheeps crashing horns on a rock mosh pit in only 48 hours for the Hardcore Punk game jam, Wool pit took the first place on this event as the fan favourite game among other submissions.

The team managed to complete the game before the jam dead line giving birth to Wool Pit (Jam version). After submission we decided to keep working on it as a complete project and the entire code was re-made to allow other features in the future.

A few weeks later we finished the mobile port, which in addition to the jam version features, includes language localization with more than 10 translations, a custom touch controller, responsive UI, an entire menu with animations and also character customization. A lot of serialization and scriptable objects were involved in this game development, which was useful for practising.
